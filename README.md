# Girl Scouts Troop Assistant

Materials, plans, and resources for a Girl Scouts troop of 8 girls, ages 6-7.

## Structure

- `meetings/` - Meeting plans and agendas
- `activities/` - Badge activities and crafts
- `communications/` - Parent emails, permission slips, schedules
- `resources/` - Templates, checklists, helpful references

## Auto-Backup

This repo is automatically committed and pushed nightly via OpenClaw cron.
